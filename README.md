## Quick Start

1. **Clone the repo**

   ```bash
   git clone https://github.com/AdilShamim8/Toolly.git
   cd Toolly
   ```
2. **Install**

   ```bash
   npm install        # or yarn install
   ```
3. **Run**

   ```bash
   toolly --help      # see available commands
   ```

That's it. You're ready to unlock your potential.
