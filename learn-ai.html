STAGE 3: AI PRACTITIONER (Weeks 13-20)

Goal: Master Prompt Engineering & AI Workflows

"The quality of AI output depends on the quality of your input"

Prompt Engineering Mastery:

Part 1: Prompt Frameworks

Framework Structure Best For
RTF Role + Task + Format General tasks
RISEN Role, Instructions, Steps, End goal, Narrowing Complex tasks
COSTAR Context, Objective, Style, Tone, Audience, Response Content creation
CREATE Character, Request, Examples, Adjustments, Type, Extras Detailed outputs
RACE Role, Action, Context, Expectation Quick tasks

Part 2: Prompt Templates Library

For Writing:

PROMPT TEMPLATE: BLOG POST

You are an expert [TOPIC] writer with 10 years of experience.

Write a comprehensive blog post about [SUBJECT].

Requirements:
- Target audience: [AUDIENCE]
- Tone: [FORMAL/CASUAL/PROFESSIONAL]
- Length: [WORD COUNT]
- Include: [SPECIFIC ELEMENTS]

Structure:
1. Attention-grabbing headline
2. Hook introduction
3. Main points with examples
4. Actionable takeaways
5. Compelling conclusion

Avoid: [WHAT TO EXCLUDE]

For Analysis:

PROMPT TEMPLATE: ANALYSIS

Act as a senior [INDUSTRY] analyst.

Analyze the following [DATA/TEXT/SITUATION]:
[INSERT CONTENT]

Provide:
1. Key findings (bullet points)
2. Patterns observed
3. Potential implications
4. Recommendations
5. Risk assessment

Format: [TABLE/REPORT/SUMMARY]

For Learning:

PROMPT TEMPLATE: LEARNING

You are a patient, expert teacher specialized in [SUBJECT].

Teach me [TOPIC] as if I were a [BEGINNER/INTERMEDIATE/ADVANCED] student.

Please:
1. Start with a simple analogy
2. Explain the core concept
3. Give 3 real-world examples
4. Provide a practice exercise
5. Summarize key points
6. Suggest next topics to learn

Use simple language. Avoid jargon unless you explain it.

Part 3: Advanced Prompting Techniques

Technique What It Does Example
Chain-of-Thought Step-by-step reasoning "Think through this step by step..."
Few-Shot Provide examples "Here are 3 examples. Now do one like this..."
Zero-Shot No examples needed "Classify this text as positive/negative"
Tree of Thoughts Multiple reasoning paths "Consider 3 different approaches..."
Self-Consistency Multiple answers, pick best "Give me 5 different solutions..."
Persona Setting Assign expert identity "You are a Harvard professor of economics..."
Constraint Setting Set boundaries "In exactly 100 words, explain..."
